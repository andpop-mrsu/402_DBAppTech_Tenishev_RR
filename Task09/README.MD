# Tic-Tac-Toe Web (SPA с Slim-бэкендом)

## Описание проекта
Веб-версия игры **Крестики-нолики (Tic-Tac-Toe)** с возможностью играть против компьютера прямо в браузере.  
Приложение реализовано как **Single Page Application (SPA)** с серверной базой данных SQLite и REST API на **микрофреймворке Slim**.

Размер игрового поля можно выбрать от 3×3 до 5×5 (внутренне поддерживается до 10×10).

---

### Правила игры
- Всегда начинают крестики.  
- Игроку случайным образом назначается символ: **X** или **O**.  
- Игрок и компьютер ходят по очереди, пока не появится победитель или поле полностью не заполнится.  
- После окончания игры результат сохраняется в базе данных **SQLite** через Slim API.  
- Для каждой партии сохраняется:
  - дата игры;
  - имя игрока;
  - фигура игрока (X или O);
  - победитель (или ничья);
  - размер поля;
  - полный список ходов.

- Можно просмотреть историю всех сохранённых игр и **воспроизвести** любую из них пошагово.

---

### Режимы работы
1. **Новая игра** — выбор размера поля и имени игрока  
2. **История игр** — таблица со списком всех партий  
3. **Воспроизведение** — анимированный просмотр выбранной игры

---

## Запуск приложения

### Требования
- **PHP 8.0 или выше** (с поддержкой SQLite и встроенным веб-сервером)  
- **Composer** (для установки зависимостей)  
- Современный **браузер** с поддержкой JavaScript-модулей

### Локальный запуск

1. **Перейдите в каталог `Task09`**
   ```bash
   cd Task09
   ```

2. **Установите зависимости через Composer**
   ```bash
   composer install
   ```

3. **Запустите встроенный PHP-сервер из корня проекта, указав `public` как корневую директорию:**
   ```bash
   php -S localhost:3000 -t public/
   ```

4. **Откройте браузер и перейдите по адресу:**
   ```
   http://localhost:3000
   ```
   или
   ```
   http://localhost:3000/index.html
   ```
