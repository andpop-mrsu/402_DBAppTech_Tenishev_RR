# Tic-Tac-Toe Web (SPA версия с серверной базой данных)

## Описание проекта
Веб-версия игры **Крестики-нолики (Tic-Tac-Toe)** с возможностью играть против компьютера прямо в браузере.  
Приложение преобразовано в **Single Page Application (SPA)** с серверной базой данных SQLite и REST API на PHP.

Размер игрового поля можно выбрать от 3x3 до 5x5 (расширяемо до 10x10).

---

### Правила игры
- Всегда начинают крестики.  
- Игрок выбирается случайным образом: крестики или нолики.  
- Игрок и компьютер ходят по очереди, пока не появится победитель или поле полностью не заполнится.  
- После окончания игры результат сохраняется в серверной базе данных SQLite.  
- Для каждой партии сохраняется:
  - дата игры;  
  - имя игрока;  
  - выбранная фигура (крестики или нолики);  
  - победитель;  
  - размер поля;  
  - список всех ходов.  
- Можно просмотреть список всех сохранённых игр и воспроизвести любую из них.

---

### Режимы работы
1. **Начать новую игру** - выбор размера поля и имени игрока
2. **Просмотреть историю игр** - список всех сохранённых игр
3. **Воспроизвести игру** - просмотр любого сохранённого матча

---

## Запуск приложения

### Требования
- **PHP 7.4 или выше** (с расширением SQLite3)
- **Веб-сервер** или встроенный PHP сервер
- **Браузер** с поддержкой ES6 модулей

### Локальный запуск

1. **Клонируйте или скачайте проект**
2. **Перейдите в каталог Task08**
   ```bash
   cd Task08
   ```
3. **Запустите встроенный PHP сервер**
   ```bash
   php -S localhost:3000 -t public
   ```
4. **Откройте браузер и перейдите по адресу**
   ```
   http://localhost:3000/
   ```
